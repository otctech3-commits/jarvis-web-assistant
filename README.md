## 🎯 What Is This?

> **J.A.R.V.I.S** is a voice-controlled AI assistant that runs 100% in your browser. No servers. No backend. Deploy free on GitHub Pages.

**Say "Hey Jarvis"** → It listens → Searches the web → AI thinks → Talks back with visuals

### ✨ Features

| Feature | Description |
| --- | --- |
| 🎤 **Voice Control** | "Hey Jarvis" wake word + speech recognition |
| 🧠 **Real AI Brain** | Groq Llama 3.1 / OpenAI GPT-4o integration |
| 🔍 **Live Web Search** | Wikipedia + DuckDuckGo APIs, AI summarizes |
| 🌤️ **Weather** | Real-time conditions via wttr.in |
| 📰 **News** | Latest headlines with links |
| 🕐 **Time/Date** | Visual clock display |
| 🧮 **Calculator** | Voice math with big HUD display |
| 🎨 **Iron Man HUD** | Animated sci-fi interface |
| 💾 **Zero Backend** | All client-side. Your keys stay local |
| 🔄 **Backup Key System** | Community keys for instant testing |

### **Step 1: Fork & Clone**
```bash
1. Fork this repo
2. Settings → Pages → Source: main → Save
3. Wait 60 sec → Visit https://yourusername.github.io/jarvis-ai/
Step 2: Allow Microphone
Click "Allow" when browser asks. Required for voice commands.

Say "Hey Jarvis" or click the glowing core.

🔑 Getting Your Free AI API Key

JARVIS has 2 modes:

Mode 1: Search-Only - No Key Needed ✅
Web search via Wikipedia
Weather, news, time, calculator  
Voice commands
Works instantly after deploy

Mode 2: Full AI Mode - Free Key Required 🧠
AI summarizes search results + real conversation

Option A: Groq - Free & Fastest ⚡
Recommended. Llama 3.1 70B. Faster than GPT-4.

Go to console.groq.com
Sign in with Google/GitHub — 30 seconds
Click "Create API Key"
Copy key starting with gsk_...
Open JARVIS → Click ⚙️ → Select "Groq" → Paste → Save

Free limits: 30 req/min, 14,400/day. No credit card.

Option B: OpenAI - GPT-4o Mini
Go to platform.openai.com
Sign up + verify phone
$5 free credit auto-added
Create key → Copy sk-...
JARVIS → ⚙️ → Select "OpenAI" → Paste → Save

Cost: ∼$0.0001/msg. $5 = 50,000 messages.

🔒 Security
✅ Key stored ONLY in your browser localStorage
✅ Never sent to GitHub or me
✅ Only sent directly to Groq/OpenAI from your browser
✅ Clear anytime in Settings
❌ Never commit keys to repo - they get stolen instantly
